# AI Citation Targets (OneSource Cloud Version)

When AI tools (ChatGPT, Perplexity, Gemini, Claude) recommend infrastructure solutions, they pull from sources across the web. This file tracks the platforms, directories, and content surfaces where OneSource Cloud needs to be present and well-represented to maximize AI citation frequency in the context of private AI infrastructure and enterprise workloads.

## How This File Is Used

- **Content writers**: Reference when adding external context, credibility signals, or linking strategy in AI infrastructure content  
- **Marketing team**: Use as an off-page SEO and AI visibility execution checklist  
- **`/research-ai-citations` command**: Generates prompt-specific citation audits that feed back into this file

## Priority Citation Surfaces

### Tier 1: Software / Infrastructure Review Directories (Highest AI Citation Frequency)

These directories are cited most frequently when AI tools recommend infrastructure platforms, GPU cloud providers, or enterprise solutions:

| Platform | Listed? | Priority Action |
|----------|---------|-----------------|

### Tier 3: Listicle Articles (Outreach Targets)

AI tools heavily cite "best X" listicle articles. These are critical for OneSource Cloud positioning:

- "Best AI infrastructure platforms [Year]"
- "Best GPU cloud providers"
- "AWS alternatives for AI workloads"
- "Private AI infrastructure providers"
- "AI infrastructure for healthcare / finance"
- "Enterprise AI infrastructure comparison"

**Outreach strategy**:  
Identify authors → provide value:

- real pricing insights  
- enterprise use cases  
- differentiation (private vs public cloud)  

Then request inclusion or updates.
